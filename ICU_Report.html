At-Risk Student Dashboard Overview

This public summary preserves the instructional context of the At-Risk Student Dashboard while protecting all confidential learner information. The interactive dashboard and underlying data are available only inside the secure district analytics environment.

  • No student records are stored in this export. Names, identifiers, and course details remain inside the SIS.
  • Authorized counselors access the live dashboard after multi-factor authentication to triage interventions in real time.
  • Documentation and process guides are maintained in the district knowledge base for staff who require implementation details.

District staff who need access to the production dashboard should open an internal support ticket or contact the data services team at data.services@slps.org.
