üîê Logging into Windows

Step-by-Step Login Process:

  1. Power On: Press the power button and wait for Windows logo
  2. Login Screen: Enter your username in the first field
  3. Password: Click password field and type your password
  4. Sign In: Click "Sign In" button or press Enter

üì∏ Using the Snipping Tool for Screen Capture

How to capture screenshots:

  1. Open Snipping Tool: Press Windows key + Shift + S
  2. Select Area: Click and drag to select what you want to capture
  3. Edit (Optional): Add annotations or highlights
  4. Save: Save the image to your desired location

Pro Tip: Use this tool to capture error messages or important information to share with tech support!

ü§ñ Using ChatGPT to Improve Your Emails

Pro Tip: You can use AI tools like ChatGPT to help proofread and improve your professional emails!

How to use ChatGPT for email proofreading:

  1. Draft your email in Outlook first
  2. Copy the text and paste it into ChatGPT
  3. Ask for help: "Please proofread this email for grammar and professionalism"
  4. Review suggestions and apply improvements
  5. Final check: Make sure the tone matches your intent

üñ•Ô∏è Understanding Outlook

Key Interface Components:

  1. Ribbon: Contains all tools and commands
  2. Navigation Pane: Switch between Mail, Calendar, Tasks
  3. Folder Pane: Shows email folders (Inbox, Sent Items, etc.)
  4. Reading Pane: Preview emails without opening
  5. Message List: Shows all emails in selected folder

‚úçÔ∏è Writing Professional Emails

Email Structure

  1. Subject: Clear, descriptive
  2. Greeting: "Dear [Name]" or "Hello [Name]"
  3. Body: Clear, concise message
  4. Closing: "Best regards" or "Sincerely"
  5. Signature: Your name and contact info
Subject: Assignment Submission - Digital Literacy

Dear Professor Smith,

I hope this email finds you well. I am writing to submit my Digital Literacy project, which is attached to this email.

Please let me know if you need any additional information.

Best regards,
[Your Name]
[Student ID]
[Email Address]

üé≠ PowerPoint Presentation Basics

Slide Design Principles:

  • 6x6 Rule: Max 6 words per line, 6 lines per slide
  • High Contrast: Dark text on light background
  • Consistent Fonts: Use 2 fonts maximum
  • Visual Hierarchy: Important items larger/bolder

Font Size Guidelines:

  • Title: 36-44pt
  • Body Text: 24-32pt
  • Use bullet points for lists

üìà Excel Spreadsheet Basics

Understanding Excel:

  1. Cells: Individual boxes (A1, B2, C3, etc.)
  2. Worksheets: Tabs at bottom for different sheets
  3. Formula Bar: Shows content of selected cell
  4. Name Box: Shows address of current cell

Common Functions:

  • =SUM(B2:B10) - Adds values
  • =AVERAGE(B2:B10) - Calculates mean
  • =MAX(B2:B10) - Finds highest
  • =MIN(B2:B10) - Finds lowest
  • =COUNT(B2:B10) - Counts entries

üìÇ File Management & OneDrive Best Practices

File Naming Conventions:

  • Descriptive names: "Biology_Lab_Report_Ch3"
  • Include dates: "Essay_Draft_2024-01-15"
  • Avoid special characters: / \ : * ? " < > |
  • Use underscores: Instead of spaces
  • Version control: _v1, _v2, _final

Folder Organization:

  • Course folders: One per class
  • Assignment types: Essays, Labs, Projects
  • Semester organization: Fall2024, Spring2024
  • Archive old work: Keep organized
