GOOGLEFINANCE Single Day Information

2. Obtain Historical Stock Information for a Single Day

Syntax: GOOGLEFINANCE(ticker, [attribute], [start_date])

Google Sheets provides historical data attributes like "open", "close", "high", "low", etc. For example, using the "all" attribute:

This will output a matrix with multiple attributes for the given start_date:

=GOOGLEFINANCE("AAPL","all",DATE(2017,2,27))
