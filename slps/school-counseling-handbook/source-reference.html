Legacy Specifics Retained Outside Handbook Body

This table documents 2017-specific content that informed the handbook but should not be copied into the handbook body as current required practice unless district leadership separately verifies it.

2017-specific contentSource locationReference handling
Board policy objectivesSource pages 141-142The 2017 policy-objective bullet list is retained as source evidence only. The handbook body uses a current-framework outcome summary and states that current board policy controls official wording.
CTE program listSource page 122The 2017 item-level program list is retained as source evidence only. The handbook body uses a career-cluster/program-category framework and requires annual verification against the current CTE catalog.
Master calendar character words and observancesSource pages 115-116The 2017 month-specific character words and named observances are retained as source evidence only. The handbook body uses generic monthly implementation windows and directs schools to current district-approved calendars.
